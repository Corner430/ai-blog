[AdGuard Home](https://github.com/AdguardTeam/AdGuardHome) 是一款全网广告拦截与隐私保护工具，通过 DNS 层面过滤广告和追踪器。本文介绍在 OpenWrt 等环境下配置 AdGuard Home 的关键步骤。

## 1 DNS 设置

![AdGuard Home 界面](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/20230604014941.png)

进入 AdGuard Home 的 Web 管理界面，点击 **设置 → DNS 设置**：

### 1.1 上游 DNS 服务器

根据网络环境填写上游 DNS，建议多填几个备用：

![上游 DNS 配置](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/20230604015138.png)

常用的上游 DNS：

- `119.29.29.29`（腾讯 DNSPod）
- `223.5.5.5`（阿里 DNS）
- `https://dns.google/dns-query`（Google DoH）
- `tls://dns.google`（Google DoT）

### 1.2 Bootstrap DNS

删除 **Bootstrap DNS 服务器**中的所有信息，然后点击应用：

![Bootstrap DNS 设置](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/1685814771128.jpg)

> Bootstrap DNS 用于解析上游 DoH/DoT 服务器的域名。如果上游 DNS 使用的是 IP 地址，可以清空此项。

### 1.3 速度限制

将速度限制改为 **0**（不限制），然后点击保存：

![速度限制设置](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/20230604015335.png)

## 2 广告过滤规则

点击 **过滤器 → DNS 黑名单**：

![DNS 黑名单](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/20230604015421.png)

点击「添加黑名单」，选择「添加自定义列表」：

![添加规则](https://cdn.jsdelivr.net/gh/Corner430/Picture/images/20230604015442.png)

### 2.1 推荐规则列表

| 规则名称       | 地址                                                             |
| -------------- | ---------------------------------------------------------------- |
| EasyList China | `https://easylist-downloads.adblockplus.org/easylistchina.txt`   |
| 广告合并规则   | `https://cdn.jsdelivr.net/gh/o0HalfLife0o/list@master/ad.txt`    |
| EasyPrivacy    | `https://easylist-downloads.adblockplus.org/easyprivacy.txt`     |
| PC 端广告规则  | `https://cdn.jsdelivr.net/gh/o0HalfLife0o/list@master/ad-pc.txt` |

更多规则可以在 Google 中搜索「adguard home 规则」获取。

---

## 3 注意事项

> **务必关闭 Turbo ACC 网络加速设置中的「DNS 缓存」和「DNS 加速」功能**，否则会与 AdGuard Home 的 DNS 功能冲突，导致广告过滤失效。
