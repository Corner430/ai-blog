- Add these lines to the _/etc/wsl.conf_ (note you will need to run your editor with sudo privileges, e.g: sudo vim _/etc/wsl.conf_):

  ```shell
  [boot]
  systemd=true
  ```

> Pay attention to port conflicts

- Reference
  [Systemd support is now available in WSL!](https://devblogs.microsoft.com/commandline/systemd-support-is-now-available-in-wsl/)
