## 1 方案一：168 元超低成本主机（E3 神教）

这套方案追求极致性价比，总花费仅 168 元，适合轻量办公、下载机、简易 NAS 或折腾学习。核心思路是利用二手服务器级 CPU（E3 系列）搭配廉价配件。

### 1.1 配置清单

| 配件     | 型号                  | 价格（元） |
| -------- | --------------------- | ---------- |
| 主板     | H61                   | 35         |
| CPU      | E3-1220 V2            | 38         |
| 内存     | DDR3 4G × 2 1600MHz   | 26         |
| 散热     | Intel 原装 / 玄冰 100 | 10         |
| 硬盘     | 120G SSD + 500G HDD   | 45         |
| 显卡     | GT240                 | 35         |
| 电源     | 180W 拆机电源         | 14         |
| 机箱     | 鞋盒手工制作          | 0          |
| **总计** |                       | **168**    |

### 1.2 注意事项

- **主板与 CPU 兼容性**：H61 主板需确认支持 Ivy Bridge 架构的 E3-1220 V2（22nm）。部分早期 H61 需更新 BIOS，购买前向卖家确认。
- **显卡选 GDDR5 版本**：GT240 有 GDDR3 和 GDDR5 两个版本，性能差距较大，务必选 **GDDR5**。
- **电源功率**：180W 对这套配置够用，注意接口是否齐全（24pin 主板 + 4pin CPU）。
- **鞋盒机箱**：注意散热通风，长期使用建议换正式机箱。

目标是组装一台多用途工作站，五系统并行不关机：Windows 11 日常办公、WSL 24 小时运行开发环境、Hyper-V 跑 NAS 和 Ubuntu Server，加上 Android 子系统。

### 2.2 三种价位组合

- **1408 元**：ECC 16G × 2 + RX 580 8G — 入门够用
- **1643 元**：ECC 16G × 2 + RX 5600 XT — 兼顾游戏
- **1989 元**：ECC 32G × 2 + RX 5600 XT — 多虚拟机最优

### 2.3 用途规划

五系统并行：

1. **Windows 11**：宿主系统，日常办公和游戏
2. **WSL**：24 小时运行开发环境
3. **Hyper-V 虚拟机 - NAS**：运行群晖等 NAS 系统
4. **Hyper-V 虚拟机 - Ubuntu Server**：跑服务和后台任务
5. **Android 子系统**：运行 Android 应用

## 3 选购建议

### 3.1 CPU

- **E3 系列**：E3-1220 V2 是 4 核 4 线程，主频 3.1GHz（睿频 3.5GHz），性能约等于 i5-3470，日常使用够用
- **E5 系列**：E5-2643 v4 是 6 核 12 线程，主频 3.4GHz（睿频 3.7GHz），适合多线程虚拟化场景。价格超过 140 元不值得入手

### 3.2 主板

- 二手主板水深，选择销量大、评价好的店铺
- X99 平台注意显卡兼容性：部分非蓝宝石/迪兰的 RX 580 可能与 X99 冲突

### 3.3 内存

- ECC 内存适合长期运行和虚拟化场景，稳定性更好
- DDR4 选择 **2R（双面）** 优于 **4R（四面）**，兼容性更好
- 预算充足直接上 32G × 2，为多虚拟机留足空间

### 3.4 显卡

- 二手矿卡需谨慎，尽量选非矿或使用时间短的
- RX 580 性价比极高，8G 显存日常使用和轻度游戏足够
- RX 5600 XT 性能大幅提升，适合有游戏需求的用户

### 3.5 电源

- 电源是整机稳定性基础，不建议过度省钱
- 方案一的 180W 仅适合低功耗配置，升级配置时需同步升级电源
- 方案二的 500W 对 E5 + RX 5600 XT 组合足够
