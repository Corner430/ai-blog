## 方案一：168 元超低成本主机（E3 神教）

这套方案追求极致性价比，总花费仅 168 元，适合用作轻量办公、下载机、简易 NAS 或折腾学习用途。核心思路是利用二手服务器级 CPU（E3 系列）搭配廉价主板和配件，以极低成本组装一台可用的主机。

### 配置清单

| 配件     | 型号                  | 价格（元） |
| -------- | --------------------- | ---------- |
| 主板     | H61                   | 35         |
| CPU      | E3-1220 V2            | 38         |
| 内存     | DDR3 4G × 2 1600MHz   | 26         |
| 散热     | Intel 原装 / 玄冰 100 | 10         |
| 硬盘     | 120G SSD + 500G HDD   | 45         |
| 显卡     | GT240                 | 35         |
| 电源     | 180W 拆机电源         | 14         |
| 机箱     | 鞋盒手工制作          | 0          |
| **总计** |                       | **168**    |

### 注意事项

- **主板与 CPU 工艺匹配**：H61 主板需要确认支持 Ivy Bridge 架构的 E3-1220 V2（22nm 工艺）。部分早期 H61 主板可能需要更新 BIOS 才能支持。购买前务必向卖家确认兼容性。
- **显卡选 GDDR5 版本**：GT240 有 GDDR3 和 GDDR5 两个版本，性能差距较大，务必选择 **GDDR5 版本**。
- **电源功率**：180W 拆机电源对于这套配置足够使用，但要注意电源接口是否齐全（24pin 主板供电 + 4pin CPU 供电）。
- **鞋盒机箱**：极致省钱方案，注意散热通风。长期使用建议还是换个正式机箱，避免积灰和安全隐患。

## 方案二：All-in-One 多用途工作站（1408-1989 元）

这套方案目标是组装一台多用途工作站，同时运行多个系统：Windows 11 用于日常办公和游戏，WSL 24 小时运行开发环境，Hyper-V 虚拟机跑 NAS 和 Ubuntu Server，再加上 Android 子系统——**五系统并行不关机**。

### 三种价位组合

- **1408 元**：ECC 16G × 2（172 元）+ RX 580 8G（308 元）— 入门够用
- **1643 元**：ECC 16G × 2（172 元）+ RX 5600 XT（655 元）— 兼顾游戏
- **1989 元**：ECC 32G × 2（406 元）+ RX 5600 XT（655 元）— 多虚拟机最优

### 用途规划

这套主机的使用方式是五系统并行：

1. **Windows 11**：宿主系统，用于日常办公和游戏
2. **WSL（Windows Subsystem for Linux）**：24 小时运行，用于开发环境
3. **Hyper-V 虚拟机 - NAS**：运行群晖等 NAS 系统，负责数据存储
4. **Hyper-V 虚拟机 - Ubuntu Server**：跑服务和后台任务
5. **Android 子系统**：运行 Android 应用

## 选购建议与注意事项

### 关于 CPU

- **E3 系列**（方案一）：E3-1220 V2 是 4 核 4 线程，主频 3.1GHz（睿频 3.5GHz），性能约等于 i5-3470，日常使用完全够用。
- **E5 系列**（方案二）：E5-2643 v4 是 6 核 12 线程，主频 3.4GHz（睿频 3.7GHz），单核性能较强，适合需要多线程的虚拟化场景。注意价格超过 140 元就不值得入手。

### 关于主板

- 二手主板水深，建议选择销量大、评价好的店铺购买
- X99 平台兼容性需要注意：部分显卡（尤其是非蓝宝石/迪兰的 RX 580）可能与 X99 平台存在兼容性问题

### 关于内存

- ECC 内存适合长期运行、虚拟化场景，稳定性更好
- DDR4 内存选择 **2R（双面）** 优于 **4R（四面）**，兼容性更好
- 如果预算充足，建议直接上 32G × 2，为多虚拟机留足空间

### 关于显卡

- 二手矿卡需谨慎，尽量选择非矿卡或矿卡使用时间短的
- RX 580 性价比极高，8G 显存日常使用和轻度游戏足够
- RX 5600 XT 性能大幅提升，适合有游戏需求的用户

### 关于电源

- 电源是整机稳定性的基础，不建议过度省钱
- 方案一的 180W 拆机电源仅适合低功耗配置，升级配置时需同步升级电源
- 方案二的 500W 额定功率对于 E5 + RX 5600 XT 组合足够使用
